# SauceDemo Testing Project

## Overview

This project demonstrates end-to-end testing for the SauceDemo application, covering both manual and automation testing practices.

## Scope

- Functional testing
- UI automation (Playwright + Gauge)
- API testing
- CI/CD integration

## Target Application

https://www.saucedemo.com/

## Tech Stack

- Playwright
- Gauge
- Postman
- GitHub Actions

## Project Structure

- docs/: Test strategy and documentation
- testcases/: Manual test cases
- automation/: Automation scripts
- bug-reports/: Bug documentation
